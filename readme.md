# Cobra

## About

This is rss reader application.

## Premise

- Python3.3.3
    - pyenv
    - Flask
    - pymongo
- mongodb

## Development

Install Python3.3 with pyenv.

```sh
$ pyenv install 3.3.3
```

Install dependency modules.

```sh
$ pip install flask
$ pip install feedparser
$ pip install pymongo
```

Install MongoDB.

```sh
$ brew install mongo
```
