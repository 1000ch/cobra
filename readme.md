# Cobra

## About

RSS Reader made with Flask+MongoDB.

[![Deploy](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy)

## Premise

- Python3.3.3
    - pyenv
    - Flask
    - pymongo
- mongodb

## Development

Install Python 3.3 with pyenv.

```sh
$ pyenv install 3.3.3
```

Install dependency modules.

```bash
$ pip install -r requirements.txt
```

Install MongoDB.

```bash
$ brew install mongo
```

Start MongoDB.

```bash
$ mongod
```
